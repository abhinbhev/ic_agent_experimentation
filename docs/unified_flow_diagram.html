Zero-shot prompting

All super-agent LLM components receive zero-shot system prompts — no domain names, no KPI references. Domain context comes only via the consolidated knowledge doc and available domains list in the human message payload.

Budget

Super-agent: 2 rounds, 3 probes/round, 5 total probes. Each probe is a full single-domain agent run, which has its own internal budget (3 rounds, 5 probes/round, 8 total retrieval calls).
